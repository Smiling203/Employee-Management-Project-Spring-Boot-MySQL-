# Project Overview
`emp-project` is a simple Employee Management System built using Spring Boot and MySQL.  
The project provides CRUD (Create, Read, Update, Delete) operations for managing employee data.

#Features
- Add a new employee
- View all employees
- View employee by ID
- Update employee details
- Delete an employee

# Technologies Used
- Backend: Java, Spring Boot, Spring Data JPA
- Database:MySQL
- Build Tool: Maven
- API Testing: Postman
- IDE: Eclipse /VS Code
